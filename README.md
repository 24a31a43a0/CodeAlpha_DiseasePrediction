# 🏥 Disease Prediction from Medical Data

### CodeAlpha Machine Learning Internship — Task 4

## 📌 Objective

Predict the likelihood of diseases based on **patient medical data** using machine learning classification algorithms.

## 🛠️ Technologies Used

* Python 3.x
* Scikit-learn
* XGBoost
* Pandas, NumPy
* Matplotlib, Seaborn

## 🤖 Models Used

|Model|Type|
|-|-|
|Logistic Regression|Linear Classifier|
|SVM (RBF Kernel)|Support Vector Machine|
|Random Forest|Ensemble Method|
|XGBoost|Gradient Boosting|

## 📁 Datasets Used

|Dataset|Disease|Source|
|-|-|-|
|Breast Cancer|Cancer (Malignant/Benign)|Sklearn built-in|
|Heart Disease|Cardiac Disease|UCI ML Repository|
|Diabetes|Pima Indians Diabetes|UCI ML Repository|

## 📊 Evaluation Metrics

* Accuracy
* Precision, Recall, F1-Score
* ROC-AUC Score
* 5-Fold Cross Validation
* Confusion Matrix

## 🚀 How to Run

1. Clone this repository

```bash
git clone https://github.com/24a31a43a0/CodeAlpha_DiseasePrediction/tree/main
```

2. Install dependencies

```bash
pip install scikit-learn xgboost pandas numpy matplotlib seaborn
```

3. Open the notebook

```bash
jupyter notebook CodeAlpha\_DiseasePrediction.ipynb
```

4. Run All Cells

## 📈 Results (Breast Cancer Dataset)

|Model|Accuracy|ROC-AUC|
|-|-|-|
|Logistic Regression|98.25% |0.9954 |
|SVM|98.25%| 0.9950|
